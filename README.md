# medicion-antenas
Este proyecto documenta el proceso de medición de parámetros de red y diagramas de radiación para dos antenas comerciales de banda ancha. El laboratorio se llevó a cabo en una **Cámara Semianecoica** utilizando instrumental de alta precisión en la materia de **Antenas y Propagación** (Optativa) en la UNRN..

## Antenas Caracterizadas
Se analizaron dos modelos específicos para cubrir un amplio rango del espectro:
- **Mini-Bicon AntennaModel 3183B:** Rango de 700 MHz a 1.1 GHz.
- **Frankonia PowerLog 70180:** Rango de 1 GHz a 18 GHz.

## Banco de Trabajo e Instrumental
La configuración del sistema de medición incluyó:
- **Analizador de Redes Vectorial (VNA) ZNC 3 VNA ROHDE&SCHWARZ:** Para la obtención del parámetro $S_{11}$ y la Carta de Smith.
- **Receptor de Test ESR EMITEST RECEIVER ROHDE&SCHWARZ:** Para la medición de potencia y diagramas de radiación.
- **Generador de Señales de RF N5181AMXG Agilent:** Fuente de señal para la antena transmisora.
- **Controlador de Mesa Giratoria:** Para realizar barridos angulares de 0° a 360° en pasos configurados.
